## Issues
If you want to submit a bug or feature request go to our [Jira issues](https://etherna.atlassian.net/jira/software/c/projects/EWEB/issues).

## Development

### 1. Install dependencies

```bash
pnpm install
```

### 2. Bootstrap CMS

```bash
pnpm bootstrap
```

### 3. Start development server

```bash
pnpm dev
```
